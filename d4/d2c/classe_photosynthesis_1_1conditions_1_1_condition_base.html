Detailed Description

template<class T, class U, MODULE ID = MODULE_NONE>
class ePhotosynthesis::conditions::ConditionBase< T, U, ID >

This template class provides a common set of funtions for every Condition subclass. This makes calling the underlying functions more straight forward.

Template Parameters
TThe class type to work with
