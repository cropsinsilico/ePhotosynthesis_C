Detailed Description

template<class T, class U, MODULE ID = MODULE_NONE>
class ePhotosynthesis::modules::ModuleBase< T, U, ID >

This template class provides a common set of static funtions for every Module subclass. This makes calling the underlying functions more straight forward.

Template Parameters
TThe class type to work with
UThe condition class associated with this Module
